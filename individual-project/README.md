## PeerLearn – Project Documentation

This repository contains the PeerLearn client (React + Vite) and server (.NET API) along with documentation.

### Quick Start

1. Ensure prerequisites: Node.js LTS + pnpm, .NET SDK, SQL Server
2. Configure environment variables for the API (see Runbook)
3. On Windows: run `start.cmd` to launch both client and server
4. On Linux/macOS: run client and server separately (see Runbook)
